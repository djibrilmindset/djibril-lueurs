# Lueurs

Jeu d'association d'idées multijoueur, jouable depuis n'importe quel téléphone.
3 à 6 joueurs, 4 manches, mécanique d'étincelles et de chutes.

## Lancement
Ouvre la page, crée un salon, partage le code à 4 caractères avec tes amis.
Chacun rejoint depuis son téléphone.

## Tech
- Single-page HTML/JS vanilla
- Supabase Realtime pour la synchro multi-device
- 60 cartes SVG procédurales générées à la volée
- Mobile-first responsive
